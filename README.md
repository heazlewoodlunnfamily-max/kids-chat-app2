## Deploy to Render (FREE)

1. Go to github.com
2. Create new repo: `kids-chat-app`
3. Upload these files:
   - server.js
   - package.json

4. Go to render.com
5. Click "New Web Service"
6. Connect GitHub
7. Select this repo
8. Build: `npm install`
9. Start: `npm start`
10. Deploy!

Wait 2-3 minutes and you'll get a URL like:
`https://kids-chat-app-xyz.onrender.com`

Share that URL with the kids - they can message from their iPhones!
